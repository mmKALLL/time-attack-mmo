Classes, Roles & the Fusion Web

Four base jobs, each branching into 3 second classes that shift into distinct MMO roles — 12 in total. A character levels two complete Beginner → 1st → 2nd paths, then fuses their two second classes into a 4th class layer (66 combinations). The merge grants that duo's 2–3 specialized skills on top of each parent's 2–3 guaranteed — a 6–9 skill core — and you keep every skill earned along both paths. Ascended forms extend the same rule later.

Rules that shape the skills

Brutal status effects
Poison ≈ 10% max HP/round. Each active status raises the target's incoming crit chance — stacking spirals fast.
Telegraphed AoE
Enemy area attacks warn 1–3 rounds ahead. Movement is free (not turn-bound), so you dodge by walking out — unless too many foes are stuck to you.
Ranged openers
Enemies own invisible approach tiles (wall-collided, not unit-collided). Archers strike while foes spend turns closing in. Attacks fire in straight lines only.
Rogue stacking
Rogues fire multiple attacks per round — 2 at first job, 3 from second job on. Mixed close/far range.
1.5s auto-cast
Both sides use their active skill every 1.5s (square timer). Some skills spend limited uses before cooldown; some cooldowns tick only while selected.
Enemy focus
Outside telegraphs, an enemy keeps hitting the nearest player it first engaged, from any of 8 adjacent directions.
